# v2.6.1

本版本优化长表格与 Mermaid 图表在所见即所得模式下的自适应布局，并修复吸顶表头和代码行号的越界问题。

## 表格布局

- 表格短列不再保留过大的统一最小宽度，存在空闲空间时会优先收窄长文本列，减少不必要的横向滚动。
- 表头保留单行最小宽度，例如“是否在 graph 中”不会为了给其他列让位而拆成两行。
- 吸顶表头按内部表格的真实边界定位，并继承原表头字体与字号；横向滚动或列宽紧凑时，吸顶列仍与正文表格对齐。

## 图表与代码行号

- Mermaid 等窄图表不再被强制放大到正文宽度，而是按渲染器给出的原始尺寸居中显示；超出正文宽度的图表仍会自动缩小。
- 修复长文档滚动后，离屏代码块的个别行号得到巨大负偏移并漂移到 Mermaid 图表等无关内容上的问题。

## macOS 安装提示

当前构建使用免费 ad-hoc 签名，暂未使用 Apple Developer ID 签名和公证。将 AnnotaMD 拖入“应用程序”后，如 macOS 阻止启动，可执行：

```bash
xattr -cr /Applications/AnnotaMD.app
```

也可以在“系统设置 → 隐私与安全性”中选择“仍要打开”。
