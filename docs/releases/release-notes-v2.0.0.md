# AnnotaMD v2.0.0

AnnotaMD V2 从原生阅读器升级为本地优先、具有飞书文档体验的所见即所得 Markdown 编辑器，并继续围绕“人通过批注给 Agent 提修改意见”这一核心工作流演进。

## 重点更新

- **全新快编辑内核**：默认直接编辑渲染后的 Markdown，源码模式保留在“视图”菜单中。
- **飞书式文档界面**：三栏布局、块菜单、选区浮动工具栏、现代文件树和紧凑设置页。
- **本地批注**：支持选区批注和全文批注，被批注文本可定位到右侧批注；批注不会写入 Markdown 渲染正文。
- **Agent 协作基础**：批注当前保存在本地 JSON，后续将迁移到 SQLite 并提供更标准的本地 Agent 接入。
- **多文件与多工作区**：支持同时打开多个文件夹、多个 Markdown 标签页，并恢复工作区状态。
- **代码块升级**：语言选择、编程语言图标、复制、全局自动换行、默认行号与更清晰的代码块外观。
- **图表升级**：Mermaid 图表支持视图切换、全屏查看、缩放和背景色设置。
- **表格升级**：窄窗口可横向滚动，滚动长表格时标题行保持吸顶。
- **统一品牌**：应用界面、安装包和仓库文档统一使用 AnnotaMD 名称。

## macOS 安装提示

当前构建暂未使用 Apple Developer ID 签名和公证。将 AnnotaMD 拖入“应用程序”后，如 macOS 阻止打开，请执行：

```bash
xattr -cr /Applications/AnnotaMD.app
```
