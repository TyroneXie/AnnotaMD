# v2.6.0

本版本新增可持久化的图片缩放与图片块交互，并统一链接点击、表格字号和图片视觉反馈。

## 图片缩放与图片块菜单

- **四角缩放**：悬浮图片后，四个角均可按原比例缩放；热区位于图片边框内，不再需要寻找远离图片的缩放光标。
- **记住图片大小**：完成缩放后，将宽度写回 Markdown 图片属性；重新打开文档时保留上次尺寸。
- **宽度吸附**：图片接近正文最大宽度时显示提示线，并自动吸附到正文宽度。
- **默认居中**：独占一段的图片默认居中，文字中的混排图片仍保持行内显示。
- **图片块菜单**：图片区域悬浮显示黄色相框图标和拖拽点；菜单固定在段落左侧，不随图片宽度移动；图片、左右留白、菜单间隙和菜单本身构成连续热区，只有移出整行才收起，并且不会切换成文字块 `T` 菜单。
- **统一操作入口**：点击图片只显示选中框与缩放能力，不再弹出顶部图片工具栏；编辑图片、行内图片、左/中/右对齐和删除图片全部移入竖排图片块菜单，删除图片固定在最后一组；再次点击已激活的“行内图片”可退出行内状态并恢复居中块图片。
- **轻量选中态**：取消图片四周的深色阴影和整体压暗，仅保留贴合图片的浅色边框。

## 链接与字体一致性

- 渲染后的链接显示手型光标，普通单击即可打开网页、应用内 Markdown 文件或文档锚点，不再要求 Cmd/Ctrl-click。
- 链接仍使用原有安全跳转链路：网页交给系统浏览器，本地 Markdown 在 AnnotaMD 内打开。
- “设置 → 编辑器 → 字体大小”同步控制正文、表格文字、数字、日期和行内代码，不再出现表格数字偏小。

## macOS 安装提示

当前构建使用免费 ad-hoc 签名，暂未使用 Apple Developer ID 签名和公证。将 AnnotaMD 拖入“应用程序”后，如 macOS 阻止启动，可执行：

```bash
xattr -cr /Applications/AnnotaMD.app
```

也可以在“系统设置 → 隐私与安全性”中选择“仍要打开”。
