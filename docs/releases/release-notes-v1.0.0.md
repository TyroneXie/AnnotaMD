# AnnotaMD v1.0.0

AnnotaMD 首个正式版本。它是一个面向 AI 内容审阅的 macOS Markdown 阅读与编辑工具：保留 Markdown 文件本身的纯文本形态，同时提供渲染阅读、源码编辑、选中批注和复制给 AI 的闭环。

## 亮点

- **阅读与编辑一体**：在阅读模式查看渲染效果，在编辑模式直接修改 Markdown 源码，支持 `⌘S` 保存和撤销/重做
- **双击定位编辑**：阅读模式双击段落即可跳到对应源码位置，适合快速修正文案
- **选中文本浮动工具条**：支持正文/标题级别、加粗、斜体、下划线、行内代码、删除、高亮和评论
- **CriticMarkup 批注**：用纯文本 CriticMarkup 记录删除、高亮、评论等修改意见，AI agent 可以直接读取这些标注继续修改
- **复制给 AI**：一键复制带说明的 CriticMarkup 全文，方便让 AI 按批注重写文档
- **Typora 风格排版**：调整字体、行距、代码块、表格和颜色对比度，让阅读效果更安静、清爽
- **表格增强**：表格默认与正文同宽，超宽时只在表格内部横向滚动，支持拖拽列宽
- **图片和图表缩放**：图片、Mermaid、PlantUML 默认撑满画布，可用按钮或 Mac 触控板手势独立缩放，不影响整页
- **代码块优化**：显示代码语言，代码默认换行，长代码不再撑宽页面
- **大纲默认打开**：长文档打开后右侧大纲默认可见
- **窗口大小记忆**：可在设置中勾选启动时保持上次退出前的窗口大小
- **统一品牌**：应用、图标、bundle、构建脚本和 DMG 产物统一为 AnnotaMD

## 安装

1. 下载 `AnnotaMD.dmg`
2. 双击打开 DMG
3. 将 `AnnotaMD.app` 拖入 `Applications`
4. 首次打开如遇 macOS 安全提示，可右键 App 选择“打开”，或运行：

```bash
xattr -cr /Applications/AnnotaMD.app
```

## 构建说明

本版本发布产物为 universal `AnnotaMD.app` 和 `AnnotaMD.dmg`，使用本地 ad-hoc 签名，适合通过 GitHub Releases 分发。
