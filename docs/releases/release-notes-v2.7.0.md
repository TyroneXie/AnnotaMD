# v2.7.0

本版本完善批注交互、默认保存与外部文件同步，并修复表格悬浮状态和设置窗口白屏问题。

## 批注联动

- 批注继续固定在正文右侧，多条批注发生纵向碰撞时自动错开，不增加连接线干扰阅读。
- 只有悬浮或点击被批注正文、批注卡片时，关联双方才进入强调状态；首条批注不再无缘无故长期加深。
- 点击批注正文或卡片可直接建立视觉对应关系，保持正文起始线和长文阅读稳定。

## 默认保存与外部文件同步

- 所见即所得与源码模式均默认开启自动保存，并共用同一套延迟保存逻辑。
- 磁盘文件被外部工具修改后自动加载最新内容，不再弹出“是否重新加载”提示。
- 外部变化到达时会先取消尚未执行的旧内容保存任务，避免编辑器中的过期内容覆盖磁盘新版本。
- 编辑器正文字号默认调整为 15px，代码块默认显示行号。

## 菜单、表格与设置

- “在上方插入 / 在下方插入”改为块与加号组合图标，方向表达更明确，并保持现有菜单图标的尺寸和粗细。
- 鼠标移动到标签栏或编辑器外部后，表格列高亮和插入线会立即清理，不再残留在文档中。
- 设置窗口等待内容完成首次绘制后再显示，修复偶发的持续白屏。
- 文件排序入口移到左侧文件栏，滚动条显隐移到“视图”菜单，常用操作无需再进入设置查找。
- 设置页面改为更紧凑的分组与间距，说明文字和控件统一左右对齐；字号、行高使用减号/数值/加号步进器。
- 自动保存延迟改为内部合理默认值，不再暴露难以判断的调节项；同时移除自动保存开启时意义有限的“保存全部”入口。
- 浅色与深色主题可直接选择，菜单名称去除多余装饰符号。

## macOS 安装提示

当前构建使用免费 ad-hoc 签名，暂未使用 Apple Developer ID 签名和公证。将 AnnotaMD 拖入“应用程序”后，如 macOS 阻止启动，可执行：

```bash
xattr -cr /Applications/AnnotaMD.app
```

也可以在“系统设置 → 隐私与安全性”中选择“仍要打开”。
