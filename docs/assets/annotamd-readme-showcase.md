# AnnotaMD

本地 Markdown 的现代写作与审阅工作流。

> 文件始终保存在本地，正文保持标准 Markdown。批注独立存储，既不污染内容，也方便 AI Agent 精准读取和处理。

## 1. 一个文档，完成写作、审阅与协作

| 写作                           | 审阅                         | 自动化                          |
| ------------------------------ | ---------------------------- | ------------------------------- |
| 所见即所得编辑，保留源码模式   | 选区批注、全文批注、正文联动 | 自动保存、外部变更自动重载      |
| 表格、任务列表、代码与 Mermaid | 批注固定右侧，碰撞自动避让   | 本地 Agent 可读取批注并修改文档 |

### 1.1. Markdown 依然开放，编辑体验更现代

```ts
const review = await annotamd.readComments(documentPath)
await agent.updateMarkdown({ documentPath, review })
```

```mermaid
flowchart LR
  A[本地 Markdown] --> B[AnnotaMD 审阅]
  B --> C[Agent 读取批注]
  C --> D[更新原文件]
```

## 2. 为长文档准备的桌面体验

- 多工作区、标签页与文档大纲
- 表格列宽优化、吸顶表头和快捷行列操作
- 图片自由缩放、对齐与尺寸记忆
- 代码高亮、默认行号和自动换行
