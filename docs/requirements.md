# AnnotaMD — 需求文档

## 1. 产品概述

AnnotaMD 是一款 macOS 原生应用，用于浏览和阅读 Markdown 文件。采用 SwiftUI + cmark-gfm + WKWebView 构建，提供三栏布局：左侧目录树导航 + 中间 Markdown 渲染显示 + 右侧大纲导航，遵循 macOS 标准设计风格。灵感源自 Buddy macOS 设计。

## 2. 目标用户

- 需要本地浏览 Markdown 文档的开发者和技术人员
- 需要阅读本地知识库/笔记的用户
- 需要快速预览 Markdown 文件的用户

## 5. 约束

- 技术栈：SwiftUI + cmark-gfm + WKWebView（v2.x 从 Textual 迁移）
- 仅支持 macOS 平台
- 只读浏览，不支持编辑
- 不做同步/云端功能
- Swift 6.0 + macOS 14.0+，使用 Swift Observation `@Observable`
- 支持多窗口打开路由
- Universal binary（arm64 + x86_64）
